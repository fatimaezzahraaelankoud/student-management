# Student Management App

Application CRUD pour gérer des étudiants, développée avec :

- Backend : Spring Boot + MongoDB  
- Frontend: React.js  
- Base de données : MongoDB



#  Fonctionnalités

- Ajouter un étudiant (Nom, Adresse, Numéro de téléphone)  
- Modifier un étudiant  
- Supprimer un étudiant  
- Lister tous les étudiants
